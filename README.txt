# HAR-Tubitak

Bu proje, insan aktivite tanıma (Human Activity Recognition - HAR) üzerine yapılmıştır. Projede kullanılan veri seti, çeşitli insan aktivitelerini kapsayan hareket verilerinden oluşmaktadır. Bu veri seti, tarafımca toplanmış ve işlenmiştir.

## Proje Hakkında

Bu proje, giyilebilir sensörler yardımıyla insan aktivitelerini tanımayı amaçlamaktadır. Projede kullanılan sensörlerden elde edilen veriler, makine öğrenimi algoritmaları kullanılarak analiz edilmiş ve sınıflandırılmıştır.

### Kullanılan Teknolojiler

- JavaScript
- Html
- Websocket
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow/Keras

### Veri Seti

Veri seti, 5 farklı spor hareketini (High knees, jumping jacks, push ups, sit ups, squat) içeren sensör verilerinden oluşmaktadır. Bu veri seti, tarafımca toplanmış olup, proje kapsamında kullanılmak üzere düzenlenmiştir.
